# Airbnb Data Analysis Project

## Project Overview
This project involves **exploratory data analysis (EDA)** on Airbnb listings data. The goal is to uncover insights about pricing, availability, reviews, and location trends to help hosts optimize their listings and understand the market.

## Dataset
The dataset includes features like:
- `id` – Listing identifier  
- `name` – Listing title  
- `host_name` – Name of the host  
- `neighborhood` – Location  
- `price` – Listing price  
- `minimum_nights` – Minimum stay  
- `number_of_reviews` – Total reviews  
- `availability_365` – Days available per year  
- `reviews_per_month` – Average monthly reviews  
- `latitude` & `longitude` – Location coordinates  

**Source:** Public Airbnb dataset from [Kaggle](https://www.kaggle.com/).

## Project Objectives
- Clean and preprocess the data (handle missing values and outliers).  
- Perform EDA to identify patterns and correlations.  
- Visualize insights with plots, heatmaps, and maps.  
- Analyze pricing trends, review distribution, and seasonal availability.  
- Optional: Predict listing prices using regression models.

## Tools & Technologies
- **Python**  
- Libraries: `pandas`, `numpy`, `matplotlib`, `seaborn`, `python`  
- **Jupyter Notebook**  

## Key Findings
- Pricing varies significantly across neighborhoods.  
- Listings with more reviews often have higher availability and pricing.  
- Certain areas show seasonal booking trends.  
- A small portion of listings contribute to most reviews.
